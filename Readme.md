

##Extracting .eml Files Using Power Automate and Azure Function

This repository provides a complete solution to automate the extraction and parsing of .eml email files using Power Automate, Azure Functions, and Python. It is designed for Information Security Analysts and developers who want to build a scalable solution for email processing.



Ensure .eml files are properly encoded before sending to Azure Function.
Use logging in Azure Function for debugging.
Secure your HTTP endpoint with API keys or OAuth.
